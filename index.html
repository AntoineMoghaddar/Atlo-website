Acknowledge The Little Ones

ATLO is an unique VR experience which allows you to learn more about bacteria. You find yourself in a home-like environment in which you can look around, teleport to different rooms and touch several objects. Information about the bacteria living on that object will pop-up whenever you touch it. In this way you can learn more about bacteria in a fun way.

Go and explore ATLO!
